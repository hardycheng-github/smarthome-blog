# 居家WiFi系列：Reolink E1 Pro 5MP 可移動AI攝影機

![](attachments/reolink_e1_pro/1.jpg)

## 快速總結

Reolink E1 Pro 可移動AI攝影機

- 5MP 超高清室內雲台攝影機
- PTZ支援 355° 水平與 50° 垂直旋轉
- AI 智能偵測（人類、動作、寵物、嬰兒哭聲）與自動追蹤功能。

**接入 Home Assistant：**

可透過 Reolink 官方整合或 ONVIF 整合接入 Home Assistant，支援影像串流、移動偵測、物件偵測、PTZ控制等功能。

建議使用 Reolink 白金級官方整合，若硬體版本不支援，可透過 ONVIF 協議作為備選方案。

接入前需在 Reolink App 先做設定配對（WiFi連接、攝影機登入帳密設定）。

支援 2.4GHz/5GHz 雙頻 Wi-Fi 或有線網路連接，建議使用 5GHz Wi-Fi 或 Ethernet 以獲得穩定的串流品質。

## 👍 優點

### 5MP 超高清畫質

提供 2880×1616（5MP）解析度，主串流可達 20fps。

### 全方位旋轉監控

支援 355° 水平與 50° 垂直旋轉，實現（幾乎）無死角監控。

支援自動追蹤功能（僅水平方向），並可設定多個預設位置，具備斷電記憶功能。

### AI 智能偵測

內建 AI 演算法，能夠識別人形、動作、寵物和哭聲。

### 雙向語音通話

內建麥克風與揚聲器，支援即時雙向語音通話，方便遠端溝通。

### 雙頻 Wi-Fi 與有線網路支援

支援 Wi-Fi 2.4GHz 或 5GHz 頻段無線連接。

同時具備 RJ45 100Mbps 乙太網路連接埠，可選擇有線連接以獲得更穩定的網路品質。

### 夜視功能

配備紅外線 LED 燈，夜視環境自動切換IR，確保夜間監控效果。

### 多種儲存選項

支援最高 512GB 的 MicroSD 卡，提供充足的本地儲存空間。

### 自動追蹤與預設位置

支援水平方向自動追蹤功能，可自動跟隨移動目標。支援最多 64 個預設位置與 1 個守衛位置，具備斷電記憶功能，斷電後仍能記住最後位置。

### 隱私模式

支援隱私模式功能，可透過軟體控制關閉攝影機，保護個人隱私。

## 👎 缺點

### 需插電使用

非電池供電設計，安裝時需考慮電源配置，無法隨意移動位置。

> 雞蛋裡挑骨頭，攝影機極少有長時間電池供電版本

### 不支援 PoE

雖然具備乙太網路埠，但無法透過乙太網路供電（PoE），需使用 DC 5V電源供電。

### Home Assistant PTZ 控制

目前最大的不便之處就是 PTZ 控制

透過 Reolink 整合的PTZ上下左右觸發後不會停止

直到移動到底或主動按下`PTZ停止`按鈕。

### 容易混淆的舊版本硬體

目前已知 Reolink E1 Pro 早期的版本（4MP或以下）

是無法直接支援加入 Home Assistant Reolink 官方整合

但是現在沒有市售這種型號的 E1 Pro

如果是在二手交易市場的話才要注意硬體版本問題

請認明 `5MP或以上`的解析度規格才能支援 Home Assistant Reolink

### 產品特色

Reolink E1 Pro 是一款專為室內監控設計的雲台攝影機，採用 1/2.7" CMOS 感光元件，提供 2880×1616（5MP）的超高清畫質。

產品支援 355° 水平與 50° 垂直旋轉，能夠實現全方位監控，無死角覆蓋。內建 AI 演算法，能夠識別人形、寵物和哭聲，並支援水平方向自動追蹤功能，減少誤報，提升監控效率。

**主要規格：**
- **解析度**：2880×1616（5MP）@20fps
- **鏡頭**：f=4.0mm fixed, F=1.6
- **主串流**：2fps - 20fps（預設 20fps），碼率 2048Kbps - 5120Kbps（預設 3072Kbps）
- **子串流**：4fps - 15fps（預設 10fps），碼率 128Kbps - 1024Kbps（預設 1024Kbps）
- **夜視**：10 顆 850nm 紅外線 LED，距離可達 12 米
- **電源**：DC 5.0V⎓1A
- **網路**：IEEE 802.11 ac/a/b/g/n（2.4GHz/5GHz）或 10M/100Mbps 乙太網路
- **儲存**：支援最高 512GB MicroSD 卡
- **尺寸重量**：Φ84×118mm，235g
- **工作環境**：-10°C~+55°C，濕度 20%~85%
- **保固**：2 年有限保固

**智能功能：**
- 支援動作偵測、人形偵測、寵物偵測、哭聲偵測
- 支援自動追蹤（僅水平方向）
- 支援最多 64 個預設位置與 1 個守衛位置
- 具備斷電記憶功能
- 支援隱私模式
- 支援縮時攝影
- 錄製模式：動作觸發（預設）、定時錄製、24/7 錄製
- 最多支援 12 個同時串流（2 個主串流 + 10 個子串流）

### 接入 Home Assistant

Reolink E1 Pro 可透過兩種方式接入 Home Assistant：

#### 方式一：Reolink 官方整合（推薦）

1. **確認硬體版本相容性**

   E1 Pro 為 5MP 版本，理論上應支援 Reolink 官方整合。若遇到相容性問題，請確認攝影機韌體是否為最新版本。

2. **在 Home Assistant 中安裝 Reolink 整合**

   - 進入 Home Assistant 設定 → 裝置與服務
   - 點擊「新增整合」
   - 搜尋「Reolink」
   - 選擇 Reolink 整合並安裝

   - 輸入攝影機的 IP 位址
   - 輸入使用者名稱和密碼（預設為 admin/admin）
   - 確認連線設定

   [需要附圖：Reolink 整合設定畫面，包含 IP、帳號密碼輸入欄位]

4. **驗證功能**

   整合完成後，您應該能在 Home Assistant 中看到攝影機的實體，包括：
   - 影像串流
   - 移動偵測
   - 人形偵測
   - PTZ 控制（平移、傾斜、縮放）

   [需要附圖：Home Assistant 中顯示的攝影機實體列表]

   - 啟用 RTSP 和 ONVIF 功能

### 注意事項

#### 硬體版本相容性

部分舊版 E1 Pro 可能無法直接與 Home Assistant 整合，建議購買前確認硬體版本。若您的攝影機不支援直接整合，可透過 Reolink NVR 或 Home Hub 連接至 Home Assistant。

#### RTSP/ONVIF 設定

使用 ONVIF 整合時，請確保：
- 攝影機與 Home Assistant 位於同一網路中
- RTSP 和 ONVIF 功能已正確啟用
- 防火牆設定允許相關連接埠通訊

#### 網路設定建議

- 建議使用 5GHz Wi-Fi 頻段或乙太網路以獲得更好的串流品質
- 確保網路頻寬充足，避免串流延遲（主串流預設碼率約 3Mbps）
- 攝影機支援多種協議：RTSP、RTMP、ONVIF、UPnP 等
- 若遇到連線問題，可嘗試將攝影機設定從 TCP 更改為 HTTP

#### 常見問題排除

**問題：無法在 Home Assistant 中看到攝影機**

- 確認攝影機硬體版本是否支援
- 檢查 RTSP/ONVIF 功能是否已啟用
- 確認 IP 位址、帳號密碼是否正確
- 檢查網路連線是否正常

**問題：影像串流延遲或卡頓**

- 檢查網路頻寬是否充足
- 嘗試切換到 5GHz Wi-Fi 頻段
- 降低串流解析度設定

**問題：PTZ 控制無法使用**

- ONVIF 整合可能無法完全支援 PTZ 功能
- 建議使用 Reolink 官方整合以獲得完整功能
